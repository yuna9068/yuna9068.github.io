MySQL 命令

2017-08-18
MySQL, 臺北科大Java

SHOW DATABASES
查詢目前伺服器現存的所有資料庫


CREATE DATABASE 資料庫名稱
建立一個新的資料庫


USE 資料庫名稱
指定預設存取的資料庫


SOURCE 路徑\檔案名稱
執行一個SQL命令檔，路徑不要有中文


SHOW TABLES
列出預設資料庫中所有資料表


DESCRIBE 資料庫名稱 / DESC 資料庫名稱
列出指定資料表的欄位資訊


TEE 路徑\檔案名稱
留存執行結果，存成文字檔，結束留存時輸入NOTEE
